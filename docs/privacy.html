Privacy Policy

Last updated: March 12, 2026

Debug Helper is a Chrome extension built by Vibery Studio. This policy explains what data the extension handles and how.

What data is captured

When you start a recording session, Debug Helper captures the following from the active browser tab:

  • DOM events (clicks, inputs, scrolls, form submissions)
  • Console output (errors, warnings, logs)
  • Network requests (URLs, status codes, timing, response bodies for failed requests)
  • Screenshots of the visible tab (when manually triggered by you)
  • Manual notes you type during the session

Where data is stored

All data stays on your device. Debug Helper stores captured data using:

  • chrome.storage.local — for session metadata and events
  • IndexedDB — for screenshot images

No data is sent to any server, cloud service, or third party. There is no analytics, telemetry, or tracking of any kind.

Sensitive data handling

Debug Helper automatically redacts sensitive data before storing it:

  • Bearer tokens and Authorization headers
  • API key patterns (sk-, ghp_, pk_live_, etc.)
  • Password field values

Redaction happens at capture time, so exported reports never contain raw credentials.

What data is exported

When you export a report (Markdown, JSON, or TOON format), the extension generates a file from locally stored data. You control where this file goes — it is not uploaded anywhere automatically. If you share the exported report (e.g. paste into an AI assistant), that sharing is your action and governed by the recipient's privacy policy.

Permissions and why they are needed

  • activeTab — capture the visible tab for screenshots
  • scripting — inject content scripts to record DOM events, console output, and network requests
  • tabs — read the current tab URL and title for report context
  • storage — store session data locally on your device
  • sidePanel — display the live event feed side panel
  • alarms — schedule event buffer flushes and storage cleanup

Data retention

Session data is stored locally until you delete it or until automatic cleanup removes old sessions (triggered when storage exceeds 80% capacity). You can delete any session manually from the extension's session history.

Children's privacy

Debug Helper is a developer tool and is not directed at children under 13.

Changes to this policy

If this policy changes, the updated version will be published at this URL with a new "Last updated" date.
